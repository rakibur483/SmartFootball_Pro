# SmartFootball Pro 

SmartFootball Pro is a Flutter-based mobile application designed for football players to track training, monitor performance progress, and manage personal profile data efficiently.

---

##  Features

* Firebase Authentication (Login & Signup)
* User Profile Management
* Training Module (Add, Edit, View Training)
* Injury Module (Awareness & Prevention)
* Progress Tracking System
* Search & Filter Trainings
* Real-time Data Update (StreamBuilder)
* Cloud Firestore Database Integration
* Delete Saved Progress

---

## Tech Stack

* Flutter
* Dart
* Firebase Authentication
* Cloud Firestore

---

## ⚙️ Setup Instructions

1. Clone the repository:
   git clone https://github.com/rakibur483/SmartFootball_Pro.git

2. Go to project folder:
   cd SmartFootball_Pro

3. Install dependencies:
   flutter pub get

4. Setup Firebase:

* Add your google-services.json
* Configure Firebase project

5. Run the app:
   flutter run
